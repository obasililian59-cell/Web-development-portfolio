# Web-development-portfolio

This repository contains my frontend development portfolio, a collection of structured, responsive web projects built as I transition into professional frontend development.

Each project is maintained in its own repository and linked from the main portfolio site to ensure modular organization and independent version control.

About Me :

I am a frontend developer in training with a background in Agricultural Science. My transition into tech reflects my interest in structured problem-solving, clean design systems, and user-focused interfaces.

Alongside development, I write fiction and maintain a creative writing portfolio, which is also structured as a standalone project.
